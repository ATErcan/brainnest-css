CSS

1. What are the main differences between external, internal, and inline CSS?

The place that we write our css code is the main difference. But this also affects the issue of which style code is applied.

2. What is the syntax for class and ID selectors?

Dot(.) for classes (.class), hash(#) for id's (#id)

3. How would you apply a single rule to two different selectors?

We select both elements with selectors and seperate them with comma, then we can give any rule we want.

4. Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?

#title.primary

5. What does the descendant combinator do?

Selects the every element inside of parent element. Doesn't matter how nested element is.

6. Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?

Class

7. From inside to outside, what is the order of box-model properties?

Content, padding, border, margin

8. What does the box-sizing CSS property do?

Specifies the behavior of width and height properties.

9. What is the difference between the standard and alternative box model?

The difference between standard and alternative box models is how they handle padding and borders. Standard box-model doesn't include paddings and borders in width and height properties. On the other hand, alternative box-model does include.

10. Would you use margin or padding to create more space between 2 elements?

Margin

11. Would you use margin or padding to create more space between the contents of an element and its border?

Padding

12. Would you use margin or padding if you wanted two elements to overlap each other?

Margin

13. What is the difference between a block element and an inline element?

A block element occupies the entire line, regardless of the width of the content in it. On the other hand, inline element takes up as much space as the content inside of it.

14. What is the difference between an inline element and an inline-block element?

Inline elements do not take properties such as margin, padding from top or bottom. Animations do not work on these elements. But these properties work for inline-block elements.

15. Is an h1 block or inline?

Block

16. Is button block or inline?

Inline

17. Is div block or inline?

Block

18. Is span block or inline?

Inline

19. What’s the difference between a flex container and a flex item?

A flex container is the element that takes the display-flex property. It is not aligned, it contains elements that are aligned. Flex-item, on the other hand, is the aligned element inside the flex-container.

20. How do you create a flex item?

Giving display property which takes flex value to it's parent element.

21. What are the 3 values defined in the shorthand flex property?

flex-grow, flex-shrink, flex-basis

22. How do you make flex items arrange themselves vertically instead of horizontally?

With flex-direction property which takes column or column-reverse value.

23. What is the difference between justify-content and align-items?

As default justify-content aligns elements horizontally and align-items aligns elements vertically. However flex-direction property can change the axis and if that happens, those 2 properties switch roles.

24. How do you use flexbox to completely center a div inside a flex container?

With giving flex-container the following properties: display: flex; justify-content: center; align-items: center;

25. What’s the difference between justify-content: space-between and justify-content: space-around?

space-between leans the first and last elements to the edges and aligns the other elements with equal spaces. space-around puts some space between the first-last element and the edges close to them, and aligns them with the other elements by keeping the space a little wider.
